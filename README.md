# ThughUnit - Modern Unit Converter
A sleek, dark-themed, feature-rich unit converter with real-time conversion and modern UI.

## Features

### ✨ Core Features
- **Real-time Conversion** — Results update instantly as you type (with smart debounce)
- **Beautiful Dark Modern UI** built with CustomTkinter
- **Swap Units** button with neon styling (`⇅`)
- **Recent History** — Last 5 conversions with click-to-restore functionality
- **Persistent Selection** — Remembers your last From/To units for each category
- One-click copy result with visual feedback
- Clean number formatting (no annoying scientific notation)
- Full keyboard support (`Enter` to log, `Ctrl+C` to copy)

### 📏 Supported Categories (8 Categories)
- **Length**
- **Weight**
- **Temperature** (with physical validation for Kelvin)
- **Volume**
- **Speed**
- **Area**
- **Pressure**
- **Time**

### 🔧 Additional Features
- Scientifically accurate conversion factors
- Smart error handling and user-friendly messages
- Custom window icon and header logo support
- Fully responsive to user input
