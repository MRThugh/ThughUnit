# ThughUnit - Modern Unit Converter

A sleek, dark-themed unit converter with a beautiful modern interface.

## Features

- Stunning dark-mode UI built with CustomTkinter
- Four essential conversion categories:
  • Length
  • Weight
  • Temperature
  • Volume
- Scientifically accurate conversion factors
- Physical validation (e.g., Kelvin cannot be negative)
- One-click copy result to clipboard with feedback
- Custom window icon and header logo support
